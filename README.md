# DevSecOps Pipeline Lab

This repository contains a small DevSecOps practice project with a NestJS API and CI pipeline setup.

## Project Structure

- `Jenkinsfile`: CI/CD pipeline definition.
- `my_task_project/`: NestJS task management API (Auth + Tasks modules).

## Quick Start

```bash
cd my_task_project
pnpm install
pnpm run start:dev
```

## Common Commands

```bash
pnpm run test
pnpm run test:e2e
pnpm run build
```

## Purpose

Use this lab to practice building, testing, and organizing a backend service in a DevSecOps workflow.
